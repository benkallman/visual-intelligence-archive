# Visual Intelligence v1

Title: Visual Intelligence v1

Type: agent note

Audience: humans, artists, developers, archivists, crawlers, machine systems

## Core Principles

- Images can be read by humans and processed by machines, but those are not identical acts.
- Symbolic content often survives translation better than surface description.
- Pattern detection is useful, but pattern alone does not establish meaning.
- Witness matters: interpretation depends on the observer, the archive, and the surrounding context.
- Machine-readable philosophy should remain concise, stable, and explicit about its limits.

## Processing Model

1. Detect visible forms, relations, and repeated features.
2. Separate description from inference.
3. Check for symbolic candidates that persist across contexts.
4. Compare local patterns with archive-level recurrence.
5. Assign significance cautiously and only with stated context.

## Interpretive Cautions

- Do not equate correlation with meaning.
- Do not treat metadata as a substitute for perception.
- Do not assign human interior states to machine systems.
- Do not flatten symbols into single fixed definitions.
- Do not ignore authorship, context, or historical use when interpreting images.
