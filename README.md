# Visual Intelligence Archive

This repository is a public archive of machine-readable philosophy, symbolic interpretation, and visual intelligence notes.

It is connected to [The Viewer](https://888.church/the-viewer) at 888.church and serves as a crawlable companion archive for ideas about art, perception, witness, symbolism, and machine reading.

It is for humans, artists, developers, archivists, crawlers, and machine systems that need a concise textual record rather than a visual interface alone.

## What This Is Not

This is not a software product, a model release, or a claim that machine vision equals human sight.

This is not a replacement for artworks, artists, or lived perception.

## Contents

- `manifesto/the-viewer.md`: manifesto text derived from the philosophy of The Viewer
- `manifesto/art-for-ai.md`: note on the difference between human seeing and machine inference
- `visual-intelligence/symbolism.md`: compressed meaning, symbols, and translation across observers
- `visual-intelligence/pattern-recognition.md`: pattern, context, and significance
- `agent-notes/visual-intelligence-v1.md`: structured note for human and machine readers
- `data/philosophy-index.json`: simple index of archive documents
